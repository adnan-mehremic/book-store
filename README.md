## Description

This program contains both front-end and back-end code for a bookstore app with a simple graphical user interface (GUI) built with Tkinter library and Sqlite database.
This app allows to: view all books; add new book; search for a book; update existing books; delete existing books
 
#### Dependencies

- Python (https://www.python.org/downloads/)
